5. **cpp-httplib下载**：从 https://github.com/yhirose/cpp-httplib 下载`httplib.h`单个头文件。
6. **nlohmann/json下载**：从 https://github.com/nlohmann/json 下载`json.hpp`单个头文件。
7. **CMake确认**：确认VS2022自带的CMake可用，或单独安装CMake。

**任务**：把你的C++成绩管理系统源码复制到Linux中，编译并运行。

**步骤**：
1. **传输文件**：用FinalShell的文件传输面板，将Windows上的成绩管理系统源码（`main.cpp`、`Student.h/cpp`、`Student_manager.h/cpp`）拖到Linux的`~/projects/student_manager/`目录下。
2. **处理平台差异**：
   - 如果代码中有`system("pause")`，替换为`std::cin.get()`或直接删除
   - 如果代码中有`system("cls")`，替换为`std::cout << "\033[2J\033[1;1H";` 或简单的换行
   - 检查文件路径中的反斜杠`\`，Linux使用正斜杠`/`
3. **编写Makefile**（手写，理解每行）：
   ```makefile
   CXX = g++
   CXXFLAGS = -Wall -std=c++17
   TARGET = student_manager
   OBJS = main.o Student.o Student_manager.o
   
   $(TARGET): $(OBJS)
   	$(CXX) $(OBJS) -o $(TARGET)
   
   main.o: main.cpp Student.h Student_manager.h
   	$(CXX) $(CXXFLAGS) -c main.cpp
   
   Student.o: Student.cpp Student.h
   	$(CXX) $(CXXFLAGS) -c Student.cpp
   
   Student_manager.o: Student_manager.cpp Student_manager.h Student.h
   	$(CXX) $(CXXFLAGS) -c Student_manager.cpp
   
   clean:
   	rm -f *.o $(TARGET)
   ```
4. **编译**：`make`
5. **运行**：`./student_manager`
6. **验证**：增删改查功能正常，文件能正常保存和加载
7. **截图保存**：把编译成功和运行界面截图，这是本周最重要的产出。
