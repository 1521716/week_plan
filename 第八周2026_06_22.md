**Part 2：按键消抖彻底搞懂（30分钟）**

按键消抖的逻辑拆解：

```text
检测按键是否按下：
  1. if(KEY == 0)           // 第一次检测到低电平
  2. Delay(20ms);           // 等20ms跳过抖动期
  3. if(KEY == 0) {         // 再次检测，确认是真的按下
  4.     // 执行按键操作
  5.     while(KEY == 0);   // 等待按键释放（防止按住时重复触发）
  6.     Delay(20ms);       // 释放也有抖动，再消一次
  7. }
```

### Day 5：数码管入门——静态显示（周五，6月26日）

【任务 | 90-120分钟】

今天开始进入新模块。数码管是51单片机必学的外设，也是密码锁的升级方向（用数码管显示输入状态）。

跟学视频：江协科技第7集（静态数码管显示）

**Part 1：理解数码管原理（30分钟）**

数码管本质上是8个LED按特定形状排列（7段+1个小数点），共阳极所有LED正极连在一起，共阴极所有LED负极连在一起。

你必须要搞懂的问题：
- 你的开发板上的数码管是共阳极还是共阴极？（看原理图或问AI）
- 段码表是什么？为什么共阳极和共阴极的段码表互为按位取反？
- 74HC138译码器在你的开发板上做什么用的？（3个IO口控制8个数码管的位选）

**Part 2：让数码管显示数字（60分钟）**

跟着视频，写出让一个数码管依次显示0-9的程序。

核心代码你最终要能独立写出：
```c
// 共阴极段码表（0-9）
unsigned char code smg_duan[] = {
    0x3F, // 0: 00111111
    0x06, // 1: 00000110
    0x5B, // 2: 01011011
    0x4F, // 3: 01001111
    0x66, // 4: 01100110

    0x7D, // 6: 01111101
    0x07, // 7: 00000111
    0x7F, // 8: 01111111

// 显示数字n

// 位选由138译码器控制（具体看你的开发板原理图）
```

**练习**：修改代码，让数码管显示你的学号后4位（每个数字停留1秒）。

### Day 6：数码管进阶——动态扫描（周六，6月27日）

【任务 | 90-120分钟】

跟学视频：江协科技第8集（动态数码管显示）

**Part 1：理解动态扫描原理（20分钟）**

问题：4个数码管显示"1234"，但只有一套段码线（8根），怎么让4个数码管显示不同数字？

答案：轮流显示。第1ms显示第1个数码管"1"，第2ms显示第2个"2"……循环足够快（>50Hz），人眼看到的就是"同时显示1234"。

**Part 2：实现动态显示（70分钟）**

跟着视频写代码，实现4个数码管同时显示不同数字。

你一定会遇到的坑：**鬼影（消影）**。切换到下一位之前，如果段码先变了而位选还没切，上一位会短暂显示下一位的数字。

解决方法（必须理解为什么这样做）：
```c
P0 = 0x00;        // 先关闭段码（消影）
位选 = 下一位;     // 再切换位选
P0 = 段码[下一位]; // 最后送新段码
```

**练习**：让4个数码管显示"1234"，同时用按键控制计数（按一下+1，数码管从0000显示到9999）。

**今日警戒线**：动态扫描跑通，消影逻辑理解到位。

**交付物**：`05_SMG_Dynamic/`目录，包含动态显示和按键计数两个程序。
