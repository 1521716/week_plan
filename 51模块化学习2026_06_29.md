**任务清单**：

**任务1.1（30分钟）**：看视频第9集，只看讲解部分，不动手。重点听讲师怎么分析"按下瞬间"和"按下状态"的区别。

**任务1.2（45分钟）**：独立写出按键状态机框架，不许看视频和之前的代码。

要求：封装一个函数 `unsigned char Key_Scan()`，返回值：
- 0 = 无按键按下
- 1 = 检测到一次有效按下（按下瞬间触发一次，按住不放只触发一次）

核心骨架：
```c
unsigned char Key_Scan() {
    static unsigned char last_state = 1;  // 上一次读取的状态
    unsigned char curr_state = KEY;       // 当前状态
    
    if(last_state == 1 && curr_state == 0) {  // 检测下降沿
        Delay(20ms);                          // 消抖
        if(KEY == 0) {                        // 确认
            last_state = 0;
            return 1;
        }
    }
    if(curr_state == 1) {                     // 按键释放
        last_state = 1;
    }
    return 0;
}
```

**任务1.3（45分钟）**：用刚封装的Key_Scan函数实现三个功能：
- 按一下按键，LED状态翻转（Toggle）
- 按一下按键，数码管数字+1（0→1→2...→9→0）
- 按一下按键，LED流水灯方向反转（本来从左到右，按一下变成从右到左）

**任务1.4（30分钟）**：改造Key_Scan，增加长按检测。

要求：按键持续按下超过2秒，返回2（长按事件）。

提示：检测到下降沿后，不是立刻返回，而是用Delay循环计时，如果2秒内按键释放了就是短按，2秒后还没释放就是长按。

**合格标准**：Task1.3的三个功能全部独立写出并编译通过。Key_Scan函数能区分"无按键/短按/长按"三种情况。

unsigned char MatrixKey_Scan() {
    unsigned char row, col;
    // 逐行扫描
    for(row = 0; row < 4; row++) {
        // 当前行输出低电平，其他行输出高电平
        P1 = ~(1 << row);  // 假设行线接P1.0-P1.3
        
        // 读取列状态（假设列线接P1.4-P1.7）
        col = (P1 >> 4) & 0x0F;

            col = (P1 >> 4) & 0x0F;
            if(col != 0x0F) {

            }

**任务清单**：

**任务4.1（30分钟）**：看视频第14集，理解定时器的硬件原理。

你必须要能回答的问题：

- 16位定时器最大能计多少个数？（65536个）

**任务4.2（60分钟）**：跟着视频配置定时器0，实现50ms精确定时。

核心配置（每一行必须理解）：
```c
// 定时器0初始化
TMOD &= 0xF0;   // 清除定时器0的模式位
TMOD |= 0x01;   // 定时器0工作模式1（16位定时器）
TH0 = (65536 - 50000) / 256;  // 初值高8位
TL0 = (65536 - 50000) % 256;  // 初值低8位
ET0 = 1;        // 打开定时器0中断
EA = 1;         // 打开总中断
TR0 = 1;        // 启动定时器0

// 中断服务函数
void Timer0_ISR() interrupt 1 {
    TH0 = (65536 - 50000) / 256;  // 重新装载初值
    TL0 = (65536 - 50000) % 256;
    // 每50ms进来一次
}
```

**初值计算公式**（必须能手算）：
- 定时时间 = 机器周期 × (65536 - 初值)
- 初值 = 65536 - (定时时间 / 机器周期)
- 例：定时50ms，机器周期1μs → 初值 = 65536 - 50000 = 15536

- 每个字节后跟一个应答位（ACK/NACK）
