## 每日作战指令

### Day 1：重置日——配置全新的Ubuntu环境（周一，6月15日）

**【数学复习 | 60分钟】**

**【主线：环境重置 | 120分钟 | 今日核心任务】**

目标：得到一个干净、标准化、可直接用于暑假开发的Linux环境。

1. **创建新虚拟机**：在VMware中新建虚拟机，安装Ubuntu 22.04 LTS。
2. **系统初始化**：
   ```bash
   # 通过FinalShell连接后
   sudo passwd root          # 设置root密码
   sudo apt update           # 更换国内源后更新
   ```
3. **搭建开发环境**：
   ```bash
   sudo apt install build-essential g++ make cmake git
   g++ --version             # 验证安装
   cmake --version
   ```
4. **创建工作区**：
   ```bash
   mkdir -p ~/projects
   mkdir -p ~/linux_practice
   ```
5. **安装文本编辑器**：
   ```bash
   sudo apt install nano     # 或 sudo apt install vim
   ```

**今日警戒线**：新虚拟机创建成功，FinalShell可以连接，`g++`和`cmake`已安装。

**【辅线：AI辅助学习 | 20分钟】**

问AI：“请解释Linux中`apt`包管理器和`build-essential`软件包的作用。”
