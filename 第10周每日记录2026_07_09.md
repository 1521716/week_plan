## 🔧 晚间学习：51 单片机定时器模块收官

完成 **简易时钟** 项目（定时器_learn/简易时钟/）：

- **硬件**：STC89C52RC + LCD1602
- **核心技术**：Timer0 1ms 中断（TH0=0xFC, TL0=0x66）→ 1000 次计数 = 1 秒
- **功能**：LCD1602 实时显示 时:分:秒（24 小时制），自动进位
- **项目结构**：main.c + LCD1602.c/h + Timer0.c/h + Project.uvproj

### 关键代码逻辑

```c
// Timer0 中断服务函数
void Timer0_Routine() interrupt 1 {
    static unsigned int Timer0_Count = 0;
    TL0 = 0x66;  TH0 = 0xFC;   // 手动重装初值 → 1ms

    Timer0_Count++;
    if (Timer0_Count >= 1000) {  // 1 秒到
        Timer0_Count = 0;
        Sec++;
        if (Sec >= 60) { Sec = 0; Min++; }
        if (Min >= 60) { Min = 0; Hou++; }
        if (Hou >= 24) { Hou = 0; }
    }
}
```
